### 使用方法

Grabcube 的界面包含“下载”、“转录”、“播放器”和“设置”四大模块，操作指引如下：

#### 1. 视频与音频下载

- **操作步骤**：
  1. 打开“下载”模块。
  2. 复制视频或音频链接（如YouTube视频）。
  3. 粘贴链接并点击“分析”。
  4. 选择需要的格式和质量，点击“下载”，并指定保存路径。
  5. 可在“任务”面板查看下载进度，完成后文件自动存储。

- **注意事项**：
  - 保证网络稳定以避免中断。
  - 支持批量下载，可逐条粘贴链接。
  - 下载过程完全本地化，无数据上传。

#### 2. AI语音转文字

- **操作步骤**：
  1. 在“转录”模块选择或上传文件。
  2. 选择目标语言（支持中、英、日、韩等）。
  3. 点击“开始转录”，软件调用Whisper模型识别。
  4. 结果以文本形式呈现，支持导出为 `.txt` 或 `.srt`。

- **特色功能**：
  - 精准语句分割，确保文本逻辑清晰。
  - 多语言专门模型优化转录质量。
  - 支持实时文本校对编辑。

#### 3. 字幕下载与编辑

- **字幕下载**：
  1. 解析视频链接时勾选“下载字幕”。
  2. 选择语言并下载，字幕与视频一同保存。

- **字幕编辑**：
  1. 导入字幕文件（支持 `.srt`、`.ass` 等）。
  2. 使用编辑器修改时间轴和文本。
  3. 导出新字幕文件以供其他播放器使用。

- **小技巧**：可预览视频同步字幕。

#### 4. AI字幕翻译

- **操作步骤**：
  1. 上传字幕或选择转录文本。
  2. 选定目标语言及AI翻译引擎（支持定制API密钥）。
  3. 点击“翻译”，结果实时显示并可双语对照。
  4. 校对后导出为字幕文件。

- **特色功能**：
  - 双语字幕同步播放，便于学习和核对。
  - 实时人工校正翻译内容。

#### 5. 使用双语播放器

- **步骤**：
  1. 在播放器模块加载视频及字幕文件。
  2. 启用“双语字幕”功能，选择原文和翻译字幕。
  3. 观看时可调整字幕样式。

- **适用场景**：语言学习和跨语言内容理解。

#### 6. 设置与优化

- 在设置模块可调整：
  - 自定义下载路径。
  - 界面和转录默认语言设置。
  - 网络代理和下载速度优化。
  - 添加自定义API密钥启用翻译引擎。
