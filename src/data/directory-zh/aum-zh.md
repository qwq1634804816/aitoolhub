AUM是一款专门为企业打造的智能AI引擎，被官方称为“企业大脑”。 它能够完全在用户自己的服务器或电脑上本地运行，无需联网，确保企业数据的完全私密和安全。用户可通过对话方式，指令AUM处理和分析公司内部的各类文件、报表及数据库，快速获取所需信息、生成文档或完成数据分析。这种架构有效避免了云端AI服务可能带来的数据泄露、法规违规（如HIPAA、GDPR）和知识产权风险。

该系统支持接入多款主流开源AI模型，方便企业按需选择和部署，比如可以安装于完全物理隔离的服务器。AUM还能为不同业务团队（如人力资源部或法务部）定制专属AI模型，助力他们高效完成各自的业务任务，如快速筛选简历或合同分析。

## 功能列表

- **完全本地化部署**：支持在企业内部服务器、私有云或完全隔离的设备上安装，保障数据不离开企业网络。
- **企业级文件搜索**：可以迅速检索海量公司内部文档和私有资料，实现精准信息定位。
- **多文件协同处理**：能同时处理大批量文件，例如批量分析简历或合同，根据指令完成对应任务。
- **数据库对话分析**：允许用户用自然语言直接向数据库提问，简化数据分析流程，免除编写复杂查询。
- **定制化AI大脑**：支持为不同部门（如人事、法务、运营）构建专属AI，贴合业务流程需求。
- **工作流自动化**：能自动创建文档、起草提案、发送邮件，或将数据写入Notion等应用，提高效率。
- **支持主流开源模型**：兼容包括Llama3、[Mistral](https://www.kdjingpai.com/le-chat-mistral/)、Mixtral、DeepSeek等多款模型，企业自由选择。
- **权限管理系统**：内置基于角色的访问控制，支持细粒度权限管理。

## 使用帮助

AUM是一款本地部署的AI工具，旨在帮助企业在不外传数据的前提下，利用AI技术分析和处理内部敏感信息。

### 安装部署

AUM提供多种部署方案，满足不同企业的安全需求，且全程无需连接公共互联网：

1. **内部服务器部署（On-Premise Servers）**

   - 企业准备性能充足的服务器硬件。
   - 将AUM软件安装于该服务器。
   - 员工通过浏览器访问服务器地址使用AUM，数据处理均在防火墙内部完成。

2. **私有云部署（Private Cloud / VPC）**

   - 企业若使用阿里云、腾讯云、AWS、Azure等云服务，并拥有虚拟私有云（VPC），可将AUM部署在其中。
   - VPC环境与公共网络隔离，保证数据安全且仅授权人员可访问。

3. **物理隔离部署（Air-Gapped Infrastructure）**

   - 适用于对安全要求极高的机构，如政府、军工、金融核心部门。
   - AUM安装于完全不连网络的独立计算设备。
   - 操作者需通过物理方式接触设备，是最高级别的安全保障。

4. **多设备本地部署（Multi-Device Local）**

   - 即将发布功能，支持AUM分布式安装于多台本地设备，实现协同工作。

### 核心功能操作

AUM主要通过对话式界面使用，类似聊天机器人交互：

#### 1. 提问和搜索文件

无需逐个打开文件，直接输入问题即可快速获得答案。

- 打开AUM主界面。
- 在对话框输入查询，如：“帮我找去年项目A的所有合同”或“第三季度财务报告中销售额最高的产品？”
- AUM自动搜索相关文件并直接回应答案。

#### 2. 同时处理多个文件

HR等部门可批量筛选资料。

- 上传或指定简历文件至AUM工作区。
- 输入指令：“筛选出所有有5年以上Java经验且毕业于985高校的候选人”。
- AUM阅读并整理出符合条件的名单。法务团队可同时比对多份合同条款。

#### 3. 与数据库对话

市场经理无需SQL即可访问数据库。

- 技术人员先将销售数据库连接到AUM。
- 用户用自然语言提问，如：“上月哪个渠道带来最多新用户？”或“生成过去半年各产品销售趋势图”。
- AUM转化提问为查询指令，并以图表或文本形式展示结果。

#### 4. 创建定制化AI大脑

为特定部门构建专属AI服务。

- 管理员后台选择“创建新大脑”。
- 上传部门相关资料，如员工手册、招聘标准等。
- 部门员工基于该资料提问，得到精准合规的回答，例如询问入职流程或生成岗位描述。

## QA

1. **AUM与ChatGPT类云端AI的区别？**
   AUM所有数据均留存在企业本地，保护数据隐私和控制权；云端AI会上传数据，存在泄露和合规风险。

2. **运行AUM对硬件有什么要求？**
   支持高性能硬件，强大资源可解锁更高级AI模型，获得更快响应。

3. **AUM支持哪些AI模型？**
   支持多款主流开源大语言模型，如Mistral、[DeepSeek](https://www.kdjingpai.com/deepseek-chatshena/)、Llama3、Mixtral等，灵活选择。

4. **AUM提供免费版本吗？**
   是的，官网提供永久免费版本，无访问限制。
