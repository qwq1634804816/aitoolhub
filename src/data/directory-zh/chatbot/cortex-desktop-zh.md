Cortex Desktop 是一款在 macOS 操作系统上运行的智能助手。它通过覆盖系统层面，让用户能够在任何应用程序中即时调取 AI 功能。用户可高亮文本，提出问题或请求解释，无需切换应用程序。Cortex 采用本地运行模式，所有数据都不会上传至云端，确保隐私安全。目前处于 Beta 测试阶段，提供为期一周的免费体验，订阅费用为每月 5 美元。该软件设计轻巧，聚焦于提升现有工作流程效率，适合开发者、设计师、学生等需要跨应用操作的群体。未来版本计划支持更深层次的工作流理解和自动化任务处理。

## 功能列表

- **即时 AI 访问**：在任意应用中选中文本，即时调用 AI 获得答案或解释。
- **上下文感知**：基于当前选中内容，提供相关建议和解答。
- **本地数据处理**：所有运算均在本地设备完成，数据绝不上传云端。
- **无缝集成工作流**：无需切换工具，直接在当前应用内使用 AI。
- **跨应用支持**：兼容大多数 macOS 应用，如文本编辑器和浏览器。
- **未来自动化功能**：计划支持工作流记忆及跨应用自动化任务执行。

## 使用帮助

### 安装流程

1. **访问官网**：打开浏览器，进入官网，点击“Get Cortex”按钮。
2. **下载安装包**：选择 macOS 版本，下载名为 `CortexInstaller.dmg` 的安装文件。
3. **安装软件**：双击 `.dmg` 文件，将 Cortex 应用拖入“应用程序”文件夹。
4. **首次启动**：打开 Cortex，阅读并同意用户协议，登录后即可进入一周免费试用期。
5. **权限设置**：根据系统提示授予 Cortex 屏幕访问权限，路径为“系统设置 > 隐私与安全”，确保 AI 功能正常运行。

### 使用方法

#### 1. 即时 AI 访问

Cortex 的核心功能是在 macOS 的任意应用中调用 AI：

- **高亮文本**：在 Notes、Safari、VS Code 等应用中选中内容，触发 Cortex（默认快捷键 `Ctrl+Shift+A`，可自定义）。
- **输入问题**：弹出小窗口显示选中文字和输入框，用户可输入如“解释这段代码”或“总结这段文字”等请求。
- **获取答案**：基于上下文生成回答，结果显示于窗口中，支持复制和继续提问。
- **快捷操作**：支持翻译、改写及文本扩展，例如选中英文文本输入“翻译成中文”即可获得结果。

#### 2. 上下文感知

- **代码解释**：开发者选中代码片段，询问功能，Cortex 会详细说明代码逻辑。
- **文本总结**：学生选中论文段落，输入“总结这段内容”，得到简明摘要。
- **问题解答**：浏览器中选中复杂术语，输入“这是什么意思？”，获得通俗解释。

#### 3. 本地运行与隐私

- **数据安全**：所有内容均在本地处理，不上传云端。
- **性能优化**：内存占用低，约 50-100MB，适配中低配置 Mac。
- **离线使用**：不依赖网络连接，支持无网环境下调用 AI。

#### 4. 自定义设置

- **快捷键**：可改为 `Cmd+Shift+C` 或其他自定义组合。
- **界面调整**：调节窗口大小、透明度和主题（浅色/深色）。
- **语言偏好**：设置 AI 默认回复语言，如中文或英文。

#### 5. 订阅与试用

- **免费试用**：安装后提供一周完整功能免费试用。
- **订阅模式**：试用期后收费 5 美元/月，订阅可在官网管理。
- **取消订阅**：随时在账户设置中取消，无额外费用。

### 操作注意事项

- **系统要求**：仅支持 macOS 10.10 及以上版本，不支持 Windows 或 Linux。
- **权限管理**：必须授予屏幕访问权限，方可读取选中文本。
- **更新检查**：定期访问官网关注 Beta 版更新，修复 bug 并引入新功能。

- **未来功能预览**：官网透露将实现“语义工作流理解”，记忆编辑项目并结合场景智能推荐相关任务。比如在 VS Code 编码时，自动提示调试工具或相关文档。
