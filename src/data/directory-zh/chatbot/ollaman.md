# OllaMan：可视化管理本地大模型的桌面客户端

OllaMan 是一款专为 **Ollama** 设计的跨平台桌面图形界面（GUI）客户端，旨在让命令行（CLI）操作 Ollama 变得更加直观和易用。它提供了现代化且优雅的界面，使得用户能够像浏览应用商店一样轻松管理本地大语言模型（如 Llama 3、[Mistral](https://www.kdjingpai.com/en/le-chat-mistral/)、Gemma 等）。借助 OllaMan，用户无需记忆繁复的终端命令，即可实现模型搜索、一键下载、版本切换及实时对话。软件支持 macOS、Windows 和 Linux 平台，适合注重隐私的个人用户及需要管理多个 Ollama 服务节点的开发团队，是提升本地 AI 交互效率的理想工具。

## 功能列表

- **可视化模型库**：提供类似“应用商店”的界面，允许用户搜索、预览并一键下载 Ollama 库中的多种大模型，免去手动输入 `pull` 命令的繁琐。
- **直观的聊天界面**：内置类似 [ChatGPT](https://www.kdjingpai.com/en/chatgpt-6/) 的对话窗口，支持 Markdown 渲染与代码高亮，能够顺畅与本地模型互动。
- **多模型管理**：轻松查看安装的模型列表，支持模型删除、详情查看和快速切换当前对话所用模型。
- **多服务器连接**：支持连接本地（localhost）及远程 Ollama 服务器，便于在不同算力设备间自由切换，例如在笔记本管理家用高性能台式机模型。
- **系统状态监控**：部分版本可监控模型运行时的硬件资源使用，帮助用户掌握系统负载。
- **跨平台支持**：原生支持 macOS、Windows 和 Linux，保证一致的用户体验。

## 使用帮助

OllaMan 的核心优势是将复杂的命令行操作转为简洁点击。以下是安装及使用的详细步骤，助你快速搭建本地 AI 助手。

### 1. 前置准备

OllaMan 是 Ollama 的图形客户端壳，在使用之前，请确保电脑已安装并运行了 **Ollama** 核心服务。

- **验证方式**：在浏览器访问 `http://127.0.0.1:11434/`，若页面显示“Ollama is running”，说明服务正常。若尚未安装，请访问 Ollama 官网进行下载安装。

### 2. 下载与安装

- **下载**：访问 [OllaMan 官网](https://ollaman.com/)，页面自动识别系统平台并提供对应安装包（macOS 为 .dmg，Windows 为 .exe，Linux 支持 .deb 或 .AppImage）。点击“Download”按钮获取。
- **安装**：
  - Windows：双击 .exe 文件，依照安装向导完成步骤。
  - macOS：打开 .dmg 文件，将 OllaMan 拖入“应用程序”文件夹。
  - Linux：赋予运行权限后直接运行 AppImage，或通过包管理器安装 deb 包。

### 3. 连接与配置

- **首次启动**：打开 OllaMan，程序默认尝试连接本地 Ollama 地址 `http://127.0.0.1:11434`。
- **连接状态**：查看左下角或右上角连接指示灯，绿色表示连接成功，红色表示连接异常，需确认 Ollama 服务是否启动。
- **远程连接（可选）**：若需连接局域网内其他电脑的 Ollama，进入“设置（Settings）”，在“Host”或“Server URL”中输入目标 IP 地址（如 `http://192.168.1.50:11434`），确保目标设备环境变量 `OLLAMA_HOST` 设置为 `0.0.0.0` 以开放外部连接权限。

### 4. 下载模型（如同逛应用商店）

- 访问左侧菜单的 **“Models”** 或 **“Library”** 页面。
- 在搜索框输入模型名称，如 `llama3` 或 `qwen`。
- 选择模型卡片并挑选版本参数（如 `8b`），点击 **“Download”** 或 **“Pull”** 按钮。
- 下载进度条显示完成后，模型自动出现在“已安装模型”列表。

### 5. 开始对话

- 点击左侧菜单的 **“Chat”** 图标。
- 顶部下拉菜单选择所需模型。
- 在输入框中输入问题并回车发送，AI 基于本地算力进行回答，全程无需联网，确保数据完全本地化。

## 应用场景

1. **隐私敏感的文档处理**  
在公司或个人电脑处理含敏感信息的文本（如财务报表、私人日记），配合本地模型确保数据交互完全本地完成，杜绝上传云端风险。
2. **开发者多模型测试**  
AI 开发者可快速切换不同模型（如对比 Llama3 与 Mistral），通过图形界面测试 Prompt 表现，无需反复用终端命令加载。
3. **家庭算力中心管理**  
用户可远程连接家中高性能台式机的模型，通过轻薄本操作，实现算力的远程调用和管理。

## QA

1. **OllaMan 是免费的吗？**  
是的，OllaMan 通常以开源或免费软件形式发布，用户可免费管理本地模型。
2. **安装 OllaMan 后还需安装 Ollama 吗？**  
需要。OllaMan 仅是客户端界面，依赖 Ollama 服务提供模型运行支持，可视其为“遥控器”，而 Ollama 是“电视机”。
3. **为何连接不上本地服务？**  
请确保：
  - Ollama 服务正在后台运行（任务栏有图标或终端已执行 `ollama serve`）；
  - 防火墙未屏蔽 11434 端口；
  - OllaMan 默认连接地址为 `localhost:11434` 未被修改。
4. **OllaMan 可直接运行 Python 代码吗？**  
其主要功能是对话和模型管理，虽能生成代码，但自身不具备执行 Python 代码的环境，应将代码复制到独立编辑器中运行。
