Pointer 是一个开源的 AI 聊天应用，采用 Electron、React 和 TypeScript 开发，由 experdot 维护并托管于 GitHub。该项目支持多种模型对话（包括 OpenAI GPT、Claude、DeepSeek 等），并具备智能的交叉数据分析和知识管理功能。Pointer 致力于帮助用户高效处理复杂信息，广泛适用于教育、研究、商业分析和内容创作等领域。项目拥有活跃的社区，拥有 35 个星标和 5 个分叉，代码库定期维护更新。用户可以通过简便的安装流程在 Windows、macOS 或 Linux 系统上运行，获得流畅的使用体验。


![Pointer 界面](https://repo.alisencent.com/upload/file/2026/04/02/170eb51a7fe0aa2.png)


## 功能列表

- 支持多种 AI 模型，用户可在 OpenAI GPT、Claude、DeepSeek等模型之间自由切换。
- 提供流式对话响应，实时展示 AI 的推理过程。
- 支持消息树形分支管理，便于追踪和切换不同对话版本。
- 具备全局内容搜索，支持关键词高亮显示。
- 内置 AI 交叉分析表功能，自动生成结构化的对比表格。
- 包含 AI 对象管理器，方便可视化管理和编辑复杂的数据结构。
- 支持数据导入导出，并兼容 OpenAI ChatGPT 和 DeepSeek Chat 的数据格式。
- 采用文件夹式知识管理，支持消息书签、标签及批量操作功能。


## 使用帮助

### 安装流程

要使用 Pointer，需完成以下环境配置和应用运行步骤：

1. **准备开发环境**
   - 安装 Node.js 18 或更高版本，详见 [Node.js 官网](https://nodejs.org/)。
   - 通过命令安装 pnpm 包管理器：

```bash
npm install -g pnpm
```

   - 系统支持 Windows 10 以上、macOS 10.15 以上及 Linux。
   - 安装 Git 用于克隆代码库。

2. **克隆代码库**

```bash
git clone https://github.com/experdot/pointer.git
cd pointer
```

3. **安装依赖**

```bash
pnpm install
```
等待依赖安装完成。

4. **启动开发模式**

```bash
pnpm dev
```
启动后，Electron 应用界面将打开。

5. **构建应用**
根据系统选择相应命令生成可执行文件：

- Windows:

```bash
pnpm build:win
```

- macOS:

```bash
pnpm build:mac
```

- Linux:

```bash
pnpm build:linux
```

构建结果存于项目目录的 `dist` 文件夹下。

6. **配置 AI 模型**

   - 启动应用后访问“设置”界面。
   - 配置模型参数，如名称、API 端点、访问密钥及模型标识符（如 `gpt-3.5-turbo` 或 `claude-3`）。
   - 设定默认模型并测试连接。


### 功能操作流程

Pointer 提供多项实用功能，使用方法如下：

1. **多模型对话**
   - 在主界面选择所需 AI 模型。
   - 输入问题或指令，AI 通过流式响应提供答案并实时展示推理过程。
   - 例如，输入“生成一个奇幻小说设定”，AI 将逐步创作内容并允许保存到消息树。
   - 利用消息树管理，切换对话分支，查看历史版本，支持新建分支。

2. **交叉分析表**
   - 打开“交叉分析表”模块，输入主题（如“比较不同编程语言”）。
   - AI 自动生成包含相关对比维度的结构化表格。
   - 用户可编辑表格数据。
   - 示例：输入“比较 Python 和 Java”，自动生成包含“语法复杂性”“性能”“应用场景”等维度的比较表。
   - 保存后可导出为 Markdown 或 JSON 格式。

3. **对象管理器**
   - 在模块中查看和编辑复杂数据结构，以树状图形式展示数据层级。
   - 支持节点拖放与内容编辑。
   - 示例：整理文献列表时，AI 生成包含作者、年份、主题等的树状结构。
   - 支持导出数据为 JSON 或 CSV。

4. **知识管理**
   - 通过“文件夹”视图组织对话和数据。
   - 添加书签和标签，方便快速查找。
   - 支持批量删除和移动操作。
   - 示例：为“机器学习”相关对话添加标签，配合全局搜索快速定位。
   - 数据备份与恢复功能支持本地文件保存与上传。

5. **全局内容搜索**
   - 通过搜索栏输入关键字，AI 会高亮并定位所有相关内容。
   - 例如，搜索“Python”，快速查找含该关键字的对话和数据节点。

6. **数据导入导出**
   - 在“设置 > 数据导入”上传符合 OpenAI ChatGPT 或 DeepSeek Chat 格式的 JSON 文件，自动解析。
   - 可导出兼容主流 AI 平台的 JSON 数据。


### 注意事项

- **API 配置**：请确保填写正确且可用的 API 密钥，以保证模型连接正常。
- **性能优化**：处理大量数据时，可关闭实时推理以提升速度。
- **数据安全**：建议定期备份数据，防止丢失。
- **调试技巧**：开发模式下，可开启 Electron 开发者工具查看日志，帮助问题排查。

通过上述指南，用户可迅速掌握 Pointer 的安装及核心功能，实现高效 AI 对话与数据分析应用。

## QA

1. **Pointer 支持哪些 AI 模型？**

Pointer 兼容多种主流模型，包括 OpenAI GPT、Claude 与 DeepSeek，用户可在设置中配置相应 API。

2. **是否需要付费？**

Pointer 本身为 MIT 许可的免费开源软件，但调用 AI 平台模型可能产生相应的 API 费用。

3. **如何迁移聊天数据？**

通过“设置 > 数据导入”上传 OpenAI ChatGPT 或 DeepSeek Chat 的 JSON 格式数据，Pointer 将自动解析导入。

4. **交叉分析表适用于哪些场景？**

适合学术文献比较、商业产品对比及教育知识整理等多种需求。
