> Copilot是微软开发的一款人工智能辅助工具，融合于Microsoft 365应用中，能够理解自然语言，从而加快获取信息和提升工作效率。它支持Word、Excel、PowerPoint、Outlook、Teams等多款应用，并以简洁的聊天界面为用户提供服务。
>
> 想要充分体验Copilot的强大功能，推荐在[Microsoft 365](https://m365.cloud.microsoft/)平台中使用。
>
> 国内访问推荐使用版本：[ChatGPT镜像站(国内访问GPT4系列模型)](https://www.kdjingpai.com/news/chatgptjingxiangzhan/)

Copilot是一款基于大型语言模型的Microsoft聊天机器人，能够引用信息来源，创作诗歌，以及为歌曲自动生成歌词和音乐。

它是微软用来替代已停用的Cortana的主打产品，最初以Bing Chat的形式整合于Microsoft Bing和Microsoft Edge。

这款工具运用了Microsoft Prometheus模型，基于OpenAI的GPT-4大型语言模型，并结合监督学习和强化学习技术进行调优，支持多种语言及方言的理解和交流。

Copilot采用免费与付费结合的模式，免费用户可以使用大多数功能，付费用户则享有更多高级AI服务和自定义聊天机器人的创建功能，这一付费版本被称为**Microsoft Copilot Pro**。默认聊天机器人包括标准版Copilot及专注于根据文本提示生成图片的Microsoft Designer。

## Copilot功能简介

- 直接与Copilot对话获取信息、解答疑问。  
- 在Microsoft 365应用中提高生产力，激发创造力，助力理解信息。  
- 利用Copilot完成Bing搜索、聊天与内容创作的一站式需求。  
- 在Microsoft Edge浏览网页时，享受实时AI辅助。  
- Windows 11任务栏直接访问Copilot，实现流畅的AI体验。  
- 在Visual Studio Code中通过自然语言提示辅助编写代码。  
- 使用Copilot Studio开发并应用AI驱动的搜索及协作工具，满足个人及企业需求。  
- 订阅Copilot Pro以获得更丰富的AI能力，包括定制聊天机器人和优先体验新功能。  
- 利用Image Creator根据文本提示生成图像，展现创意构思。


## Copilot使用指南

### 安装与配置

1. **安装Microsoft 365**：确保已安装Microsoft 365套件并持有效订阅。  
2. **启用Copilot功能**：在Microsoft 365应用内开启Copilot，通常位于设置或插件管理中。  
3. **登录微软账号**：使用Microsoft账户登录，实现数据与设置同步。

### 功能操作示例

#### 智能文档生成

1. 打开Microsoft Word应用。  
2. 输入希望生成内容的提示或大纲。  
3. 点击工具栏上的Copilot图标，选择“生成文档”。  
4. 查看并编辑Copilot生成的内容。

#### 演示文稿制作

1. 打开Microsoft PowerPoint。  
2. 导入已有Word文档。  
3. 选择工具栏Copilot图标中的“生成演示文稿”。  
4. 调整自动生成幻灯片并优化样式。

#### 邮件管理

1. 启动Microsoft Outlook。  
2. Copilot自动分类邮件，优先显示重要邮件。  
3. 阅读邮件时获得智能回复建议，可以直接发送或修改。

#### 数据分析

1. 启动Microsoft Excel。  
2. 导入分析数据。  
3. 通过Copilot图标启动“数据分析”功能。  
4. 查看生成的数据报告并据此做决策。

#### 任务自动化

1. 运行任意Microsoft 365应用。  
2. 设置需要自动化的任务。  
3. 使用Copilot选择“任务自动化”并启动。  
4. 实时监控自动化任务的执行情况。

#### 会议助手

1. 打开Microsoft Teams。  
2. 会议开始前启用Copilot会议助手。  
3. 会议结束后自动生成会议记录和摘要。  
4. 根据记录分配相关后续任务。
