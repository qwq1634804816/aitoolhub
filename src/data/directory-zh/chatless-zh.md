chatless 是一款轻量级且开源的 AI 聊天客户端，支持 Windows、MacOS 与 Linux 平台。它主要为用户提供本地 AI 聊天、文档解析及知识库管理功能，专为低性能设备设计，避免运行繁重的程序带来的资源压力。该项目由 kamjin3086 开发，采用 Tauri 2.0 与 Next.js 15 构建，结合简洁的界面设计，确保软件占用小、运行流畅。用户可通过本地或远程的 [Ollama](https://www.kdjingpai.com/ollama/) API 进行交流，并支持快速总结和分析知识库。项目开发进度约为 85%，未来将于 GitHub 完全开源。

![chatless截图](https://repo.alisencent.com/upload/file/2026/04/02/ac890f468058d5f.webp)

## 功能列表

- **AI 聊天**：支持本地或远程 Ollama API 对话，适合推理问题和简单代码生成。
- **本地文档解析**：解析并提取本地文档内容，支持对话式查询。
- **知识库管理**：建立和管理本地知识库，支持生成嵌入，快速总结与分析文档。
- **历史记录管理**：保存并管理聊天记录，便于回顾与整理。
- **轻量级设计**：安装包仅约 20MB，资源占用低，适合低性能设备。
- **跨平台支持**：兼容 Windows、MacOS 和 Linux 系统。
- **个性化设置**：提供灵活的配置选项，满足用户习惯。

## 使用帮助

### 安装流程

chatless 是开源项目，需要从 GitHub 克隆代码并在本地构建，步骤如下：

1. **克隆代码库**  
在终端执行命令：

```bash
git clone https://github.com/kamjin3086/chatless.git
```

此操作将项目下载到本地。注意，截至 2025 年 7 月代码尚未完整开源，可能需等待进一步发布。

2. **安装依赖环境**  

chatless 基于 Tauri 2.0 和 Next.js 15，要求安装以下环境：

- **Node.js**：推荐使用 16 及以上版本，检测命令：

```bash
node -v
```

若未安装，请访问 [Node.js 官网](https://nodejs.org/) 下载。

- **Rust**：Tauri 依赖 Rust 编译器，安装命令：

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

- **Tauri 依赖**：参考 [Tauri 官方文档](https://tauri.app/) 安装操作系统所需依赖。

3. **安装项目依赖**  
切换到项目目录：

```bash
cd chatless
```

然后安装 Node.js 依赖：

```bash
npm install
```

4. **构建与运行**  
构建 Tauri 应用：

```bash
npm run tauri build
```

运行开发模式：

```bash
npm run tauri dev
```

运行成功后，chatless 会启动并显示主界面。MacOS 安装包大小约 20 多 MB，具体因系统略有不同。

5. **配置 Ollama API**  
chatless 默认通过 Ollama 执行 AI 推理，需确保 Ollama 已安装并运行：

- 本地部署：下载并安装 Ollama，详情见 [Ollama 官网](https://ollama.com/)。
- 远程部署：在设置中填写远程 Ollama API 地址，格式如 `http://<ip>:<port>`。

在 chatless 的“AI 模型提供商设置”页面配置 API 并保存。

### 功能操作流程

#### AI 聊天

1. 打开 chatless，进入“聊天”页面。
2. 选择本地或远程 Ollama API（需事先设置）。
3. 输入问题或指令，如“帮我写一个 Python 循环”或“总结这篇文档”。
4. AI 实时响应，答案显示在对话框，支持继续提问和查看历史记录。

#### 本地文档解析

1. 在主界面选择“文档解析”。
2. 上传支持的文档格式，如 PDF、TXT、DOCX。
3. 软件自动提取内容，生成可查询的文本。
4. 输入与文档相关的问题，如“提取文档中的关键点”，AI 将基于内容回答。
5. 可将解析结果保存至知识库，便于后续调用。

#### 知识库管理

1. 访问“知识库”页面，点击“新建知识库”。
2. 上传文档或手动录入，系统将生成嵌入（支持本地或 Ollama 推理）。
3. 使用对话功能查询知识库内容，如“总结知识库中的财务数据”。
4. 在“知识库详情”中管理内容或调整嵌入参数。

#### 历史记录管理

1. 点击主界面“历史记录”选项。
2. 浏览聊天记录，支持按时间或主题筛选。
3. 支持导出为文本文件，或删除无用记录。

#### 个性化设置

1. 进入“常规设置”，调整主题、字体大小等界面选项。
2. 在“知识库设置”中配置嵌入生成方案（本地或远程）。
3. “AI 模型提供商设置”页面可切换 API 或调整模型参数。

### 注意事项

- **性能优化**：chatless 设计针对低性能设备，但需确保 Ollama 推理服务稳定。
- **文档支持**：支持大多数常见文档格式，复杂或加密文档可能需预处理。
- **开源计划**：代码尚未完全开源，建议关注 GitHub 仓库更新。
- **社区支持**：开发者欢迎反馈意见，支持在 GitHub 提交 issue。

## 应用场景

1. **个人知识管理**  
便于将学习笔记或工作资料导入知识库，快速总结与查询，适合学生和研究人员。
2. **远程 AI 推理**  
在低性能电脑通过远程 Ollama API 使用高性能服务器，适合需要处理复杂任务的专业人士。
3. **轻量级代码辅助**  
程序员可利用 chatless 生成简单代码和调试片段，界面简单且节省资源。
4. **本地化办公需求**  
企业员工可解析本地文档并与 AI 交互，数据无需上传至云端，有效保护隐私。

## 常见问答（QA）

1. **chatless 支持哪些 AI 模型？**  
当前重点支持 Ollama 的本地及远程 API，未来可能扩展其他模型。
2. **如何在低性能电脑上运行？**  
安装包体积小（约 20 MB），使用 Tauri 与 Next.js 优化性能，支持远程 API 减轻本地负载。
3. **知识库功能如何使用？**  
上传文档或手动录入内容生成嵌入，之后可通过对话查询，适合快速分析总结。
4. **是否需要联网？**  
本地模式无需联网，远程模式需连接 Ollama 服务。
