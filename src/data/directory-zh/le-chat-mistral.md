LE CHAT MISTRAL 是由法国Mistral AI公司推出的多语言对话助手平台。该平台旨在通过教育且有趣的方式，让用户与Mistral AI的多种模型互动体验。用户可根据需求选择 Mistral Large、Mistral Small，或试用简洁的原型模型 Mistral Next。目前，LE CHAT MISTRAL仍处于测试阶段，用户可自由注册并体验其功能。

## 功能列表

- 提供多种Mistral AI模型的访问入口
- 支持多语言对话交互
- 兼容 Mistral Large 和 Mistral Small 模型
- 提供正在测试的原型模型 Mistral Next
- 借助简洁的交互体验掌握前沿的AI技术


## Mistral可用模型型号

**先进**  
- Mistral Large：面向高复杂度任务的顶级推理模型，最新v2版本于2024年7月发布  
- Mistral NeMo：最佳多语言开源模型，将于2024年7月发布

**专业**  
- Codestral：专为编程任务打造的先进语言模型，预计2024年5月发布  
- Mistral Embed：用于文本摘录语义表示的前沿模型

**研究**  
- Mistral 7b：首个密集模型，已于2023年9月发布  
- Mixtral 8x7b：首个稀疏混合专家模型，预计2023年12月发布  
- Mixtral 8x22b：目前最强开源模型，于2024年4月发布  
- Mathstral 7b：首个数学开源模型，预计2024年7月发布  
- Codestral Mamba：首个mamba 2开源模型，预计2024年7月发布
