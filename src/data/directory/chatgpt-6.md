ChatGPT is a chatbot launched by OpenAI based on large language models that can generate text similar to that of humans. It can guide conversations according to user input and flexibly adjust the format, style, level of detail, and language of the replies. Thanks to its ability to generate natural feedback and coherent interactions, ChatGPT is widely popular. This website also showcases various practical application examples, including creative inspiration, task automation, and more.

# What can ChatGPT be used for?

- Generate text content similar to humans
- Control the length, format, style, richness of details, and language type of conversations
- Conduct interactive conversations on various topics
- Support diverse applications such as brainstorming names, fun event planning, proposal design, thank-you letters and email writing, travel planning, article summarization, code debugging, text editing, content optimization, and recipe recommendations
- Achieve natural and smooth conversational experience through voice assistants
- Execute specified tasks directly without entering prompts after preset commands


# ChatGPT usage help

- Enter topics or questions you care about, ChatGPT will generate targeted responses
- Guide and adjust conversation style, format, and depth through specific instructions
- Suitable for various scenarios such as email writing, creative brainstorming, and event suggestions


# ChatGPT features

## Basic features of ChatGPT

- Default support for GPT-3.5 model (testing more advanced models at times)
- Custom user instructions
- Chat history management and archiving
- Sharing conversation links

## Paid features of ChatGPT

- Includes all basic features
- Access to GPT-4 model
- GPT-4 supports DALL·E image generation
- Plugin store providing thousands of plugins
- GPTs feature for creating or using custom tools shared by the global community
- Priority access to experimental new features

## About GPTs

GPTs are innovative features that allow users to customize personalized ChatGPT versions according to their needs. By integrating prompts, external knowledge, and specialized functions, users can create GPTs applications suited for different tasks such as writing, painting, data analysis, game assistance, and negotiation, and share these applications.

Users can log in to the official ChatGPT website to create and test personal GPTs applications through the GPT Builder tool or manual configuration, and can also refer to official OpenAI examples for inspiration.

GPTs features include:

- **Prompts**: Guide GPTs' response style and task execution through keywords, questions, instructions, or examples, such as simulating celebrity tones or answering specific domain questions.
- **Additional knowledge**: Upload text or image materials to enhance GPTs' accurate information sources, such as auxiliary references for legal documents or artworks.
- **Specific functions**: Equip GPTs with advanced skills to broaden application scenarios, including:
  - **Web browsing**: Supports real-time internet search, suitable for news, weather, and stock updates.
  - **Image generation**: Create images based on descriptions or examples, suitable for art design and entertainment.
  - **Code interpreter**: Write and run Python code, generate charts, handle diverse data and files, suitable for data analysis, programming teaching, and automation.
- **Custom APIs**: Add specific interfaces or commands to meet special needs, such as querying weather, placing orders, or sending emails.

# ChatGPT Pricing

- **Free version**: Suitable for first-time individual users, includes:
  - Unlimited messages, interactions, and chat history
  - Access to GPT-3.5 model (limited to 40 conversations/3 hours)
  - Supports web, iOS, and Android platforms
  - Price: Free

- **Plus version**: Suitable for enhancing personal productivity, includes all free version features plus:
  - Access to the most powerful GPT-4 model (40 conversations/3 hours)
  - Browsing and using GPTs tools
  - Use of special tools such as DALL·E, Browsing, Advanced Data Analysis
  - Price: $20 per user/month

- **Team version**: Suitable for rapidly growing teams, improves collaboration efficiency, includes all Plus version features plus:
  - Higher message limits and GPT-4 access
  - Creating and sharing GPTs within the team
  - Admin console
  - Team data not used for training
  - Price: $25/user/month billed annually, $30/user/month billed monthly

- **Enterprise version**: For innovative enterprises supporting secure scaling, includes all Team features plus:
  - Unlimited high-speed access to GPT-4 and other tools
  - Larger context window support for long text inputs
  - SAML single sign-on (SSO)
  - Enterprise data by default not used for training, supports customizable data retention periods
  - Account management, admin controls, and domain verification features
  - Priority technical support and dedicated account managers
  - Price: contact sales team
