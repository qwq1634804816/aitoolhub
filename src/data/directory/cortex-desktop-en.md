Cortex Desktop is an intelligent assistant running on the macOS operating system. By overlaying at the system level, it allows users to instantly invoke AI functions within any application. Users can highlight text, ask questions, or request explanations without switching applications. Cortex operates locally, ensuring no data is uploaded to the cloud, thereby safeguarding privacy. Currently in Beta testing, it offers a one-week free trial and costs $5 per month after the trial period. The software is lightweight and focuses on enhancing existing workflows, making it suitable for developers, designers, students, and others who require cross-application operations. Future versions plan to support deeper workflow understanding and automated task handling.

![Feature Illustration](https://repo.alisencent.com/upload/file/2026/04/02/6b749e8d6b3adc5.png)

## Features List

- **Instant AI Access**: Select text in any application and instantly invoke AI for answers or explanations.
- **Context Awareness**: Provides relevant suggestions and answers based on the current selected content.
- **Local Data Processing**: All computations are performed locally on the device; data is never uploaded to the cloud.
- **Seamless Workflow Integration**: Use AI directly within the current application without switching tools.
- **Cross-Application Support**: Compatible with most macOS applications, such as text editors and browsers.
- **Future Automation Features**: Plans to support workflow memory and cross-application automated task execution.

## Usage Help

### Installation Process

1. **Visit the Official Website**: Open the browser, go to the official website, and click the "Get Cortex" button.
2. **Download Installer**: Choose the macOS version and download the installation file named `CortexInstaller.dmg`.
3. **Install the Software**: Double-click the `.dmg` file and drag the Cortex app into the "Applications" folder.
4. **First Launch**: Open Cortex, read and agree to the user agreement, then log in to start the one-week free trial.
5. **Permission Settings**: Grant Cortex screen access permissions as prompted by the system under “System Settings > Privacy & Security” to ensure AI features operate correctly.

### How to Use

#### 1. Instant AI Access

The core feature of Cortex is invoking AI in any macOS application:

- **Highlight Text**: Select content in apps such as Notes, Safari, VS Code, etc., and trigger Cortex (default shortcut `Ctrl+Shift+A`, customizable).
- **Input Questions**: A popup window appears showing the selected text and an input box where users can request explanations like "Explain this code" or "Summarize this text."
- **Receive Answers**: Generates responses based on context, displayed in the window with options to copy or continue asking.
- **Quick Actions**: Supports translation, rewriting, and text expansion. For example, select English text and input "Translate to Chinese" to get results.

#### 2. Context Awareness

- **Code Explanation**: Developers select code snippets and ask about functionality; Cortex provides detailed explanations of the code logic.
- **Text Summarization**: Students select sections of papers and input "Summarize this content" for concise summaries.
- **Question Answering**: Select complex terms in the browser, input "What does this mean?" to get plain explanations.

#### 3. Local Operation & Privacy

- **Data Security**: All content is processed locally without uploading to the cloud.
- **Performance Optimization**: Low memory usage, about 50-100MB, suitable for mid-to-low spec Macs.
- **Offline Use**: Does not rely on internet connection; AI can be invoked offline.

#### 4. Custom Settings

- **Hotkeys**: Can be changed to `Cmd+Shift+C` or other custom combinations.
- **Interface Adjustments**: Adjust window size, transparency, and theme (light/dark).
- **Language Preference**: Set AI default reply language, such as Chinese or English.

#### 5. Subscription and Trial

- **Free Trial**: Offers a full-featured one-week free trial after installation.
- **Subscription Model**: Charges $5/month after trial; subscriptions manageable on the official website.
- **Cancel Subscription**: Can cancel anytime in account settings with no extra fees.

### Operational Notes
