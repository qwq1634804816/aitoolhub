Voquill is an AI tool integrated into the Chrome browser that allows users to replace keyboard typing with voice input, applicable to any website text input scenario. Whether writing emails, replying to chats, or editing documents, the real-time speech-to-text feature makes it easy to complete tasks. In addition, Voquill has an AI mode where users can issue voice commands, and AI will automatically generate complete responses based on the current webpage content. This significantly improves communication efficiency for users who are not skilled typists or whose typing efficiency is affected by physical reasons.

## Features List

- **Speech-to-Text**: Instantly converts users' spoken content into text in any website input box.
- **Multi-Scenario Support**: Suitable for writing emails, social media replies, and online document editing.
- **AI Mode**: Automatically generates context-relevant replies by combining webpage environment and voice commands.
- **Dictation Mode**: Accurately converts speech into clean, well-formatted text with punctuation.
- **Browser Integration**: Available as a Chrome extension for easy and quick access anytime.

## Usage Help

Voquill is a simple-to-use Chrome browser extension whose main function is to convert speech into text, enabling users to "input by speaking without manual typing."

### Installation Process

1. **Visit the Official Website or App Store**: Open Chrome browser and go to the official website [voquill.com](https://voquill.com/) or search "Voquill" on the Chrome Web Store.
2. **Add to Chrome**: Click "Add to Chrome" and confirm the permission request in the pop-up.
3. **Confirm Installation**: Choose "Add Extension." After installation, the Voquill icon will appear in the browser toolbar.
4. **Pin the Icon**: Click the browser’s extension icon, find Voquill, and pin it for easy access.

### Core Function Operations

Voquill provides two main modes: `Dictation Mode` and `AI Mode`.

#### 1. Dictation Mode

This is the most commonly used basic function, suitable for entering any text on webpages such as writing emails or replying to messages.

- **Activation**:
  - Place the cursor in the text box where input is needed (e.g., Gmail email body or WeChat web input box).
  - Click the Voquill icon on the browser toolbar or use the shortcut key to start the microphone.
  - A visual indicator will appear when listening is active.

- **Start Input**:
  - Speak directly, keeping a steady pace and clear pronunciation.
  - Speech will be converted into text in real-time and displayed in the input box.
  - Voquill automatically punctuates and segments sentences, ensuring clean and tidy text.

- **End Input**:
  - After speaking, click the icon or shortcut again to turn off the microphone.

#### 2. AI Mode

AI Mode is Voquill’s innovative feature, which not only transcribes speech but also understands vocal instructions and automatically generates structured replies based on webpage context.

- **Activation Method**:
  - Place the cursor in the reply area, such as within an email reply.
  - Switch to AI Mode via the plugin’s interface (possibly through right-click menu or icon options).

- **Give Instructions**:
  - Use voice to issue task instructions rather than directly reading the reply content, for example, "Help me reply to this email, tell the sender I have received the file and express thanks, plan to give feedback by Friday."

- **Intelligent Generation**:
  - AI analyzes the current webpage content and the instruction.
  - Within seconds, it generates a reply text meeting the requirements, for example: "Hi [Sender's Name], thank you for your file. I have received it and will give you feedback by this Friday, thanks!"

- **Edit and Send**:
  - Users can modify the generated content and send after confirmation.

This function is especially useful for occasions requiring formal, logically clear replies, significantly saving thinking and typing time.
