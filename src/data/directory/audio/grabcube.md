### How to Use

Grabcube's interface includes four main modules: "Download," "Transcribe," "Player," and "Settings." Instructions are as follows:

#### 1. Video and Audio Download

- **Steps**:
  1. Open the "Download" module.
  2. Copy the video or audio link (e.g., YouTube video).
  3. Paste the link and click "Analyze."
  4. Choose the desired format and quality, click "Download," and specify the save path.
  5. View download progress in the "Tasks" panel; files are automatically saved upon completion.

- **Notes**:
  - Ensure a stable network to avoid interruptions.
  - Supports batch downloads by pasting links one by one.
  - The entire download process is fully localized with no data uploaded.

#### 2. AI Speech-to-Text

- **Steps**:
  1. Select or upload files in the "Transcribe" module.
  2. Choose target language (supports Chinese, English, Japanese, Korean, etc.).
  3. Click "Start Transcription," the software uses the Whisper model for recognition.
  4. Results are displayed as text and can be exported as `.txt` or `.srt`.

- **Special Features**:
  - Precise sentence segmentation ensures clear text logic.
  - Specialized multilingual models optimize transcription quality.
  - Supports real-time text proofreading and editing.

#### 3. Subtitle Download and Editing

- **Subtitle Download**:
  1. When parsing video links, check "Download Subtitles."
  2. Select language and download; subtitles are saved along with the video.

- **Subtitle Editing**:
  1. Import subtitle files (supports `.srt`, `.ass`, etc.).
  2. Use editor to modify timeline and text.
  3. Export new subtitle files for use with other players.

- **Tips**: Preview video with synchronized subtitles.

#### 4. AI Subtitle Translation

- **Steps**:
  1. Upload subtitles or select transcribed text.
  2. Choose target language and AI translation engine (custom API keys supported).
  3. Click "Translate," results display in real-time with bilingual comparison.
  4. Proofread and export as subtitle files.

- **Special Features**:
  - Bilingual subtitle sync playback facilitates learning and verification.
  - Real-time manual correction of translation content.

#### 5. Using the Bilingual Player

- **Steps**:
  1. Load video and subtitle files in the player module.
  2. Enable "Bilingual Subtitles" feature and select original and translated subtitles.
  3. Adjust subtitle styles during playback.

- **Use Cases**: Language learning and cross-language content comprehension.

#### 6. Settings and Optimization

In the Settings module you can adjust:
- Custom download paths.
- Default interface and transcription language settings.
- Network proxy and download speed optimization.
- Add custom API keys to enable translation engines.
