LE CHAT MISTRAL is a multilingual conversational assistant platform launched by the French company Mistral AI. This platform is designed to enable users to interact with various Mistral AI models in an educational and entertaining way. Users can choose from Mistral Large, Mistral Small according to their needs, or try the streamlined prototype model Mistral Next. Currently, LE CHAT MISTRAL is still in the testing phase, and users are free to register and experience its features.

Free API: [No need to deploy large models locally anymore, Mistral offers free API packages for developers, accessible directly from within China!](https://www.kdjingpai.com/buyaozaibendebuan/)

## Features List

- Provides access to multiple Mistral AI models
- Supports multilingual conversational interaction
- Compatible with Mistral Large and Mistral Small models
- Offers the prototype model Mistral Next currently under testing
- Enables mastering cutting-edge AI technology through a streamlined interactive experience


## Mistral Available Model Versions

**Advanced**  
- Mistral Large: Top-tier reasoning model for highly complex tasks, latest v2 version released in July 2024  
- Mistral NeMo: Best multilingual open-source model, scheduled for release in July 2024

**Professional**  
- Codestral: Advanced language model specifically designed for programming tasks, expected release in May 2024  
- Mistral Embed: Cutting-edge model for textual excerpt semantic representation

**Research**  
- Mistral 7b: First dense model, released in September 2023  
- Mixtral 8x7b: First sparse mixture of experts model, expected release in December 2023  
- Mixtral 8x22b: Currently the strongest open-source model, released in April 2024  
- Mathstral 7b: First mathematics open-source model, expected release in July 2024  
- Codestral Mamba: First mamba 2 open-source model, expected release in July 2024
