Luobi AI Writing is an intelligent assistant tool focused on novel and online literature creation. It integrates various advanced AI models including Gemini, ChatGPT, Claude, and Kimi, providing full-process support from inspiration generation, outline planning, character creation to main text writing and polishing. Creators can start writing in multiple ways based on their habits, such as inputting ideas, outlines, or imitating excellent openings. The platform can generate multiple chapters at once, supports continuous creation, and intelligently links previously set characters and worldviews to ensure story coherence. Moreover, Luobi AI offers intuitive table and list format outline editing to help users clearly organize story structure. It also allows importing external texts for style analysis and imitation, assisting users to draw inspiration and improve writing skills.

## Features List

- **Multi-model Support**: Compatible with various AI large models including Gemini, ChatGPT, Claude, DeepSeek, Kimi, Doubao, Zhipu, Grok, etc.
- **Diverse Creation Processes**: Support starting writing from an opening, outline, or imitation of openings.
- **One-click Generation and Imitation**: Quickly generate or imitate short stories, introductions, and openings.
- **Multi-chapter Continuation**: Capable of continuing multiple chapters at once and pre-generating story development ideas.
- **Outline Visualization Editing**: Edit outlines, scenes, characters, and plots in table or list format for better clarity.
- **Intelligent Linking**: Automatically connect characters, worldviews, and preceding content to ensure text coherence.
- **Main Text Optimization**: Provide multi-dimensional AI optimization suggestions with one-click adoption for revisions.
- **Setting Management**: Support visual editing and importing of external worldviews and character settings.
- **AI Book Analysis and Imitation**: Support uploading `.txt` and `.docx` files, automatically parse chapters for analysis and imitation.
- **Inspiration Generation**: Rich built-in prompt templates to quickly spark writing inspiration.
- **Prompt Memory**: Save frequently used prompt templates and automatically record the last used state for convenient access.

## Usage Guide

Luobi AI Writing is an online platform that requires no installation and can be used directly through the official website. Below are instructions for main functional modules:

### 1. Start a New Book Creation

The platform offers flexible starting methods for different users:

1. **Visit Official Website**: Open a browser and go to `https://luobi.net`.
2. **Enter the Creation Interface**: Click the "Enter Luobi" or "Start Creating" button.
3. **Choose a Startup Method**:
   - **Start from an Opening**: Input existing opening text for AI to expand creation.
   - **Start from an Outline**: Use the visual outline editor to create main and subplot lines and key nodes, supporting both table and list modes.
   - **Imitate an Opening**: Upload or paste sample text; AI analyzes the style and generates a similar opening.

### 2. Core Writing Features

1. **Generate and Continue Multiple Chapters**
   - After finishing the opening or outline, choose "Generate Multiple Chapters at Once" or "Continue Multiple Chapters".
   - Customize the number of chapters needed.
   - AI automatically links existing characters, worldviews, and previous content to ensure story coherence.
   - Provides a “Generate Subsequent Plot Inspirations” feature for idea reference.
2. **Outline Editing**
   - Intuitive outline editor supports creating plot points, adding scenes, characters, and plot descriptions.
   - Supports drag-and-drop to adjust structure; all outline content drives main text generation.
3. **Character and Worldview Management**
   - Sidebar provides “Character” and “Settings” management interfaces.
   - Support creating detailed character information such as names, personalities, appearances, and backgrounds.
   - The settings page allows importing external files and editing worldviews, geography, history, and traits.

### 3. Content Optimization and Analysis

1. **Main Text Optimization**
   - After completing a chapter draft, select paragraphs or the entire text to call "Optimize Text".
   - AI offers grammar, expression, rhythm, and other modification suggestions, supporting item-by-item acceptance.
   - One-click optimization automatically adjusts the content.
2. **Book Analysis and Imitation**
   - Upload files in `.txt` or `.docx` format; the system automatically splits chapters.
   - Input custom commands; AI analyzes book structure, character arcs, and pacing.
   - Based on the analysis, achieve work imitation and skill learning.

### 4. Prompts and Inspiration

1. **Inspiration Generation**: Built-in diverse prompt templates help overcome writing block.
2. **Prompt Memory**: Support customizing and saving frequently used prompt templates; the system records the last usage status for easy recall.
