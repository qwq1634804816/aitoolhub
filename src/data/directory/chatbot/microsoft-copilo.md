> Copilot is an AI assistant developed by Microsoft, embedded in Microsoft 365 applications. It comprehends natural language to accelerate information retrieval and improve work efficiency. It supports Word, Excel, PowerPoint, Outlook, Teams, and other applications, providing service to users through a simple chat interface.
>
> To fully experience the powerful features of Copilot, it is recommended to use it on the [Microsoft 365](https://m365.cloud.microsoft/) platform.
>

Copilot is a Microsoft chatbot based on large language models, capable of citing sources, creating poetry, and automatically generating lyrics and music for songs.

It is Microsoft’s flagship product to replace the discontinued Cortana, initially integrated within Microsoft Bing and Microsoft Edge as Bing Chat.

This tool uses Microsoft’s Prometheus model, based on OpenAI’s GPT-4 large language model, and is fine-tuned with supervised learning and reinforcement learning techniques. It supports understanding and communication in multiple languages and dialects.

Copilot adopts a freemium model, where free users can access most functionalities, and paid users enjoy more advanced AI services and the ability to create customized chatbots. This paid version is called **Microsoft Copilot Pro**. Default chatbots include the standard Copilot and Microsoft Designer, which focuses on generating images based on text prompts.

## Copilot User Guide

### Installation and Configuration

1. **Install Microsoft 365**: Ensure Microsoft 365 suite is installed and you have a valid subscription.  
2. **Enable Copilot Feature**: Activate Copilot within Microsoft 365 apps, typically found in settings or plugin management.  
3. **Sign in with Microsoft Account**: Log in using your Microsoft account to synchronize data and settings.

### Feature Operation Examples

#### Intelligent Document Generation

1. Open Microsoft Word application.  
2. Enter the prompt or outline for the content you want to generate.  
3. Click the Copilot icon in the toolbar and select "Generate Document."  
4. Review and edit the content generated by Copilot.

#### Presentation Creation

1. Open Microsoft PowerPoint.  
2. Import an existing Word document.  
3. Select "Generate Presentation" from the Copilot icon in the toolbar.  
4. Adjust auto-generated slides and optimize styles.

#### Email Management

1. Launch Microsoft Outlook.  
2. Copilot automatically categorizes emails, prioritizing important messages.  
3. Receive intelligent reply suggestions when reading emails, which can be sent directly or edited.

#### Data Analysis

1. Launch Microsoft Excel.  
2. Import data for analysis.  
3. Start the "Data Analysis" function via the Copilot icon.  
4. Review generated data reports and make decisions accordingly.

#### Task Automation

1. Run any Microsoft 365 application.  
2. Set tasks that require automation.  
3. Use Copilot to select "Task Automation" and start the process.  
4. Monitor the execution of automated tasks in real-time.

#### Meeting Assistant

1. Open Microsoft Teams.  
2. Enable Copilot Meeting Assistant before the meeting starts.  
3. Automatically generate meeting minutes and summaries after the meeting.  
4. Assign related follow-up tasks based on the records.
