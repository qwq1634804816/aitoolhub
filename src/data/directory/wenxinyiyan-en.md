## Usage Help

#### Chat and Q&A

- Enter questions or chat content in the main interface input box and click send; Wenxin Yiyan will respond based on the input.
- For example: Enter "How is the weather today?" to get the current weather conditions.

#### Image Recognition and Generation

- Upload an image or describe the image you want to generate; Wenxin Yiyan will complete the recognition or generation task.
- For example: Upload a landscape photo; the system will recognize and describe elements in the image.

#### Copywriting

- Enter the copywriting topic and specific requirements; Wenxin Yiyan will automatically generate the corresponding content.
- For example: Enter "Write an article about environmental protection," and you will get a related text.

#### Inspiration Provision

- Provide the theme or field for creation; Wenxin Yiyan will output multiple creative suggestions.
- For example: Enter "Design a creative concept for a new product," and receive several conceptual plans.

#### Document Reading

- After uploading a document, the system quickly reads and extracts summaries and key content.
- For example: Upload a PDF file; Wenxin Yiyan can extract the main information.

#### Intelligent Translation

- Enter the text to be translated and select the target language; Wenxin Yiyan will perform accurate translation.
- For example: Enter "Hello, how are you?" and select Chinese; it will translate as "你好，你怎么样？"

#### Code Generation

- Enter code requirement descriptions; Wenxin Yiyan will generate the corresponding programming code.
- For example: Enter "Write a web crawler program in Python," and get a Python code example.
