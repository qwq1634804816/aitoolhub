Tongyi Qianwen is an intelligent large model developed by Alibaba Cloud, relying on deep learning and natural language processing technologies to provide human-like interactive experiences. It can rapidly create copywriting, enrich life interests, and serve as a learning assistant to help users easily master various kinds of knowledge. With leading technology and continuously upgraded capabilities, Tongyi Qianwen is increasingly becoming an indispensable intelligent helper in daily work and life.

The Tongyi series of AI assistants and tools include Alibaba Cloud Bailian, Tongyi Lingma, Tongyi Xiaomi, Qianwen Large Language Model, Qianwen VL, Tongyi Wanxiang, Tongyi Tingwu, and Tongyi Zhiwen, committed to providing users with comprehensive services through general artificial intelligence.

![Tongyi Qianwen](https://repo.alisencent.com/upload/file/2026/03/31/13d11f17f01be2d.png)

## Features List

- Creative copywriting generation
- Fun lifestyle guidance
- Learning tutoring and assistance
- Intelligent interactive dialogue
- Various question answering
- Personalized recommendations and services

### Dialogue Tool

**Type `/` to quickly evoke prompt instruction templates**

![Tongyi Qianwen Multimodal Large Model](https://repo.alisencent.com/upload/file/2026/03/31/c0e2f3f8b305021.png)

**Supports document upload**

Up to 100 files can be uploaded simultaneously (single file max 150MB), supporting formats including PDF, Word, Excel, Markdown, EPUB, Mobi, and TXT.

**Image Parsing**

Supports uploading a single PNG or JPG image, up to 10MB in size.

### Efficiency Tools

- Real-time recording: speech-to-text, synchronous translation, and automatic key point summarization
- Upload audio and video: speech-to-text, automatic speaker identification, supports one-click export
- Document reading: supports uploading various documents and automatically extracts key information
- Webpage reading: add webpage links for quick content and key point summarization
- Paper reading: upload academic papers to extract the most valuable knowledge
- Book reading: import e-books and organize key points by chapter
- Podcast link transcription: input RSS subscription address, intelligently extracts and summarizes without downloading

![Efficiency Tools Interface](https://repo.alisencent.com/upload/file/2026/03/31/810f13ff07602f0.png)

**Intelligent Agent**

Simple intelligent agents can be built within the Tongyi Qianwen app, with main functions provided by the official team. Compared to platforms such as Zhipu Qingyan, Tencent MetaBot, Wenxin Intelligent Agent, and Kouzi, the intelligent agent development capability is relatively weaker.
